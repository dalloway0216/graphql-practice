# graphql-practice

Node.js + Apollo Server + GraphQL の練習プロジェクトです。

## 学習内容

- Node.js
- Apollo Server
- GraphQL Schema
- Query
- Mutation
- Resolver
- Arguments
- CRUD

## 機能

- users：ユーザー一覧取得
- user(id)：IDでユーザー取得
- addUser：ユーザー追加
- deleteUser：ユーザー削除
- updateUser：ユーザー更新

## 使用技術

- JavaScript
- Node.js
- Apollo Server
- GraphQL

## 実行方法

```bash
npm install
node index.js
```

Apollo Sandbox:

```text
http://localhost:4000
```
